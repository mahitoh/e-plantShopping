# Paradise Nursery (e-plantShopping)

A premium plant nursery e-commerce application designed to provide a serene and efficient shopping experience for indoor gardening enthusiasts.

## ✨ Features

- **Landing Page**: Immersive entry point with a "Get Started" journey and blurred background aesthetics.
- **Product Gallery**: 
  - Over 30 unique plants across 5 specialized categories (Air Purifying, Aromatic, Medicinal, Insect Repellent, Low Maintenance).
  - Categorized layout for easy discovery.
  - Dynamic "Add to Cart" functionality with state-aware buttons.
- **Real-time Navigation**: Persistent navbar with a live cart quantity badge visible on all sub-pages.
- **Advanced Shopping Cart**:
  - Global state management for persistent item storage.
  - Interactive quantity adjustments and one-click removal.
  - Subtotal and grand total auto-calculations.
  - "Continue Shopping" logic to preserve user journey.

## 🛠️ Technology Stack

- **Frontend**: React.js (Vite)
- **State Management**: Redux Toolkit & React-Redux
- **Styling**: Vanilla CSS (Responsive Design)
- **Deployment**: GitHub Pages (gh-pages)

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/mahitoh/e-plantShopping.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the development server**:
   ```bash
   npm run dev
   ```

## 📦 Deployment

This project is configured for easy deployment to GitHub Pages. Simply run:
```bash
npm run deploy
```
